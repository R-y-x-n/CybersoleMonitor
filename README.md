# Cybersole Monitor

## Getting Started

A) Clone/Download this Repo
B) Run 'npm install'
C) Edit config.json to your liking
D) Create your proxy file and either leave it empty for proxyless or add proxies
E) Run 'npm start'

### Prerequisites

You need to have Node.js installed

# Cybersole Monitor

## Getting Started

* Clone/Download this Repo
* Run 'npm install'
* Edit config.json to your liking
* Create your proxy file and either leave it empty for proxyless or add proxies
* Run 'node monitor'

### Prerequisites

You need to have Node.js installed
